Text Font Subsetting — ?subset=

Roboto font loaded with ?subset=ABCDEFGHIJKLMNOPQRSTUVWXYZabcdefghijklmnopqrstuvwxyz0123456789.,!? — only Latin letters, digits, and basic punctuation are kept.

The quick brown fox jumps over the lazy dog. 0123456789 

@font-face {
  font-family: 'Roboto Subset';
  src: url('./roboto.woff2?subset=A-Za-z0-9') format('woff2');
}

JS Import — ?subset=

Font imported via JavaScript with ?subset= query. Useful for runtime font loading (Rive, Canvas, PDF generators). 

import fontUrl from '@fontsource/roboto/files/roboto-latin-400-normal.woff2?subset=Hello'

Google Fonts Optimization

The plugin automatically appends &text= parameter to Google Font URLs, letting Google's servers do the subsetting. 

<link href="https://fonts.googleapis.com/icon?family=Material+Icons">
// Plugin transforms to:
// ...?family=Material+Icons&text=close+menu+search+home+...

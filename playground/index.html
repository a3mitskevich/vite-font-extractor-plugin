Safari Fix

Safari only: This feature fixes a Safari-specific rendering bug where minified icon fonts may display incorrectly. The visual difference is only visible in Safari browser. In other browsers, both columns will look identical.

Without safariFix

star favorite home search

With safariFix: true

star favorite home search 
// vite.config.js
targets: [
  { fontName: "Icons Normal",    ligatures: ["star", "favorite", "home", "search"] },
  { fontName: "Icons SafariFix", ligatures: ["star", "favorite", "home", "search"], safariFix: true },
]
